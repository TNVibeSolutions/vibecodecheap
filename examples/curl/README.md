# cURL Examples

Examples using cURL with VibeCodeCheap API.

## Configuration

Replace `your_api_key` in the scripts with your actual API key.

Or use environment variable:

```bash
export API_KEY=your_api_key
```

## Examples

### Anthropic API Format (Recommended)

| File | Description |
|------|-------------|
| `anthropic_basic.sh` | Simple messages API |
| `anthropic_streaming.sh` | Streaming response |

```bash
chmod +x anthropic_basic.sh anthropic_streaming.sh
./anthropic_basic.sh
./anthropic_streaming.sh
```

### OpenAI API Format

| File | Description |
|------|-------------|
| `basic.sh` | Simple chat completion |
| `streaming.sh` | Streaming response |

```bash
chmod +x basic.sh streaming.sh
./basic.sh
./streaming.sh
```
